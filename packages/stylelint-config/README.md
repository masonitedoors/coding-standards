# @masonite/stylelint-config

## Usage

Locally install into your project using this command:

```shell
npx install-peerdeps --dev @masonite/stylelint-config
```

Add to your project's package.json:

```json
{
  "name": "mypackage",
  "version": "0.0.1",
  "stylelint": {
    "extends": "@masonite/stylelint-config"
  }
}
```
